# Admin Guide — Screenshot capture checklist

Save each PNG into `docs/images/` with the **exact filename**. It appears in the
guide automatically (no code change). Missing files show a tidy “pending” note.
Live operational screenshots are allowed, but this guide is publicly served:
never expose customer contact details, medical notes, receipts, vaccine documents,
or other sensitive drawer content. Collapse customer rows or use a safe crop.
